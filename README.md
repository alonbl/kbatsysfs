# Sysfs Battery Plasma Widget

No need for upower/systemd and such, a simple widget to sample battery.

## Build

Refer to Makefile.

## Installation

Install packaged plasmoid into user's home by:

$ plasmapkg -i kbatsysfs-${VERSION}.plasmoid
